# 🔥 THE SHADOW CHEATS - Save File Editor

## Ultimate Binary Save Editor for Game Files

Modify your game save files directly in your browser. Works 100% offline - no data uploaded to any server.

### ✨ Features

- **Edit all game settings**: FPS, Graphics, Controls, Audio, Camera, Gameplay
- **Binary file support**: Works with original .sav files
- **No backend required**: Everything runs locally in your browser
- **Mobile friendly**: Works on phones and tablets
- **Instant download**: Get your modified file immediately

### 🚀 Live Demo

[Click here to use the editor](https://shadowsupportmail-create.github.io/ACTIVE-SAV/)

### 📱 How to Use

1. Click the upload area or drag & drop your `.sav` file
2. Edit any setting - sliders, checkboxes, number inputs
3. Click "DOWNLOAD EDITED .SAV" to save your modified file
4. Transfer back to your game folder

### 🎮 Supported Settings

| Category | Settings |
|----------|----------|
| FPS & Performance | FPSLevel, BattleFPS, LobbyFPS |
| Graphics | ArtQuality, RenderQuality, etc. |
| Sensitivity | Camera sensitivity, Gyroscope, 3D Touch |
| Audio | Master volume, Background music |
| Camera | FOV, Vehicle/Parachute camera |
| Gameplay | Item limits, Auto pickup |
| Visuals | Hit effects, Colors |

### 🔒 Privacy

**100% private** - All file processing happens in your browser. Your files never leave your device.
